# opusai_prakalp

Installation

1. Clone repository
  
2. Set up the backend:
  cd backend  
  python -m venv venv  
  venv\Scripts\activate  
  pip install -r requirements.txt  

3. Set up the frontend:  
  cd ../frontend  
  npm install  


Run the application

1. Start the backend  
  cd ../backend  
  venv\Scripts\activate  
  python main.py  

2. Start the frontend  
   cd ../frontend  
   npm run dev  

3. http://localhost:5173
